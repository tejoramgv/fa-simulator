Finite Automata Simulator - Test Page

Project location: C:\Users\Tejoram\fa-simulator

To run:

  1. Install Node.js from nodejs.org
  2. Open terminal/command prompt
  3. Run: cd C:\Users\Tejoram\fa-simulator && npm install && npm run dev
  4. Open browser to http://localhost:5173

What's included:

  • Generator - Create DFA from "strings ending with 01", "even number of 0s"
  • Converter - DFA↔NFA, ε-NFA→NFA conversions
  • Display - Interactive graph visualization
  • Regex - Regular expression ↔ NFA (Thompson Construction)
  • Simulator - Test strings with step-by-step animation
  • Minimizer - DFA minimization (Hopcroft's algorithm)
